# Kiran’s Diner — Restaurant Ordering UI

This is a small front-end project that simulates a restaurant ordering experience.
Users can add available items to their carts, increase quantity, remove items and finally submit payment through a popup modal form.

## Features:

Displays menu items dynamically from a JS data array

Clicking "+" adds the item to the cart (doubles price on repeated adds)

“remove” decreases quantity / removes item when quantity becomes 0

Total amount updates automatically

“Complete Order” shows a card payment popup modal with blurred background

“Pay” button displays a success message

Tech used:
HTML, CSS, JavaScript (no frameworks)
